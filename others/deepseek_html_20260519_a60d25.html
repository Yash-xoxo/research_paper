Week 1
📚 Dataset & literature review Days 1–5
Download blood smear / blood group datasets from Kaggle (ErythrocyteID, etc.)
Read 5–8 relevant IEEE/Springer papers; note methodological gaps
Organise dataset into class folders (A, B, AB, O) & standardise naming
Analyse class distribution; plan augmentation strategy if imbalanced
Write Introduction + Literature Review draft (~30% of final paper)
Week 1–2
🖼️ Preprocessing & augmentation Days 4–10
Resize all images to 224×224 px (standard input for CNNs)
Apply CLAHE & histogram equalisation for stain normalisation
Gaussian blur + mild denoising (OpenCV)
Augmentation pipeline: rotation, flip, zoom, brightness shift
Split data: 70% train / 15% val / 15% test; store splits with metadata
Week 2–3
🧠 Model training & evaluation Days 8–18
Build lightweight custom CNN (3–4 conv blocks) in Keras/TensorFlow
Implement transfer learning: MobileNetV2 (frozen base) + dense head
Train for 30–50 epochs with early stopping & ReduceLROnPlateau
Plot accuracy & loss curves; save best model checkpoint (.h5)
Generate confusion matrix, classification report (per-class metrics)
Week 3–4
⚙️ Optimization & explainability Days 18–24
Hyperparameter tuning: learning rate (1e-3 → 1e-5), dropout, batch size
Add Grad‑CAM visualisations to highlight RBC decision regions (novelty angle)
Report: Accuracy, Precision, Recall, F1-score, AUC-ROC
Compare custom CNN vs MobileNetV2 results (inference time + accuracy table)
Week 4
📄 Paper drafting & arXiv upload Days 24–30
Write Methodology, Results, and Discussion sections with final metrics
Use IEEE two-column conference template (LaTeX/Overleaf)
Add IEEE‑formatted citations (10–20 references from recent literature)
Preprint upload to arXiv (cs.CV or eess.IV) and share on ResearchGate
✨ Key tip: Start writing your paper from Week 1 — keep a shared document with methods, results, and citations. Document each experiment outcome immediately. This parallel workflow saves 40% of time compared to last‑minute writing.
